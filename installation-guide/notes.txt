Website: https://www.oracle.com/java/technologies/downloads/

Windows -

1️⃣ Download Java

Go to Windows section.

Download x64 Compressed Archive (.zip).

2️⃣ Extract the ZIP File

Extract it to a location like:

C:\Java\

After extraction it will look like:

C:\Java\jdk-21

3️⃣ Set JAVA_HOME Environment Variable

Click Edit the system environment variables

Click Environment Variables

Under System Variables → New

Add:

Variable Name: JAVA_HOME
Variable Value: C:\Java\jdk-21

4️⃣ Add Java to PATH

In System Variables

Select Path → Edit

Add:

%JAVA_HOME%\bin

5️⃣ Verify Java Installation

Open Command Prompt and run:

java -version

You should see something like:

java version "21"
Java(TM) SE Runtime Environment
Java HotSpot(TM) 64-Bit Server VM
